## 先搞清楚:你卡在哪一步?

订阅海外 AI,其实是**两件事**,别混为一谈:

1. **付款**:国内信用卡 / 支付宝大多付不了海外订阅(被 Stripe 风控拒),这是核心难题。
2. **访问**:网页版 / API 有时需要稳定的网络环境。

这份指南主攻**付款**(最难那块),顺带说访问。先定位你的需求,再往下看:
- 只想用 **ChatGPT/Claude 网页或 App 会员** → 看方案 ①②③
- 要付 **API、广告、App Store、SaaS** → 只能方案 ③
- 是**开发者只调 API** → 方案 ④ 最省

## 一句话先给结论

| 你的情况 | 最省心 | 最划算 | 最长久 |
|---|---|---|---|
| 只想快点用上 Plus/Pro | 代充 | 美区礼品卡(支付宝买) | — |
| 长期续费、还要付别的(API/广告/SaaS) | 虚拟卡 | 虚拟卡 | 虚拟卡 |
| 开发者,只调 API | API 中转(支付宝充值) | API 中转 | 自有海外卡 |
| 既要订阅又要访问又要 API | 一体化服务 | — | — |

## 四大方案详解

### ① 代充(最省心,但你不碰账号)
- **原理**:你付人民币,平台用官方内购通道帮你把账号升级成 Plus/Pro。
- **怎么做**:找代充店(电商 / 社群)→ 提供账号或买卡密 → 平台代充 → 账号变会员。
- **优点**:零门槛、最快、不用自己折腾支付。
- **缺点**:你**不掌握支付方式,续费要一直找人**;有溢价;鱼龙混杂,务必挑信誉好的、
  支持担保交易的。
- **适合**:只想马上用上、不想碰任何技术的人。
- **成本**:约官方价 **+ 10%~30%** 辛苦费,每次续费都溢价。

### ② 美区 App Store 礼品卡(支付宝就能买)
- **原理**:用支付宝买美区 App Store 礼品卡,充进美区 Apple ID,走 Apple 内购订阅。
  支付走 Apple **不走 Stripe**,所以**几乎零封号**。
- **怎么做(iOS)**:
  1. 注册 / 切换一个**美区 Apple ID**(地区美国,无需美国卡)。
  2. 支付宝搜「**大牌礼卡**」或「出境 / 折扣礼卡」→ 进官方合作伙伴 **Pockyt** →
     搜 **App Store & iTunes USA** → 用人民币买礼品卡。
  3. App Store 充值礼品卡余额 → 在 ChatGPT / Claude App 内购订阅。
- **安卓同理**:Google Play 订阅,支付走 Google;可在 Google 支付中心把地址设成
  **免税州**,确认 Tax = $0.00。
- **优点**:正规、安全、人民币直付、几乎不封号。
- **缺点**:有 **Apple/Google 抽成**(月成本略高于官方价);**只能付 App 内的东西**,
  付不了 API、广告、网页版 SaaS。
- **适合**:只在 App 内订阅会员、最怕封号的人。

### ③ 虚拟信用卡(最通用,但要挑还活着的)
- **原理**:开一张**美国卡段 Visa 虚拟卡**,像真卡一样付几乎所有海外服务。
- **怎么做**:选平台 → 实名 / 注册 → 开卡(通常一次性开卡费)→ 充值 → 把卡号绑到
  ChatGPT/Claude/API/广告后台 → 正常扣款。绑卡时**账单地址用美国地址**(平台一般会给)。
- **能付什么**:ChatGPT、Claude、Cursor、OpenAI/Anthropic **API**、App Store、
  **Meta/TikTok/Google 广告**、Midjourney、各种 SaaS——**一卡通付所有**。
- **优点**:最通用、自己掌握、续费不求人。
- **缺点**:这行**洗牌很快**——WildCard 2025 年 7 月已停业;**Stripe 会封特定卡段**,
  今天能用的卡段明天可能被拒;**别在平台囤太多余额**,用多少充多少。
- **目前还在运营的(以各家最新状态为准)**:BinPay、PokePay、VCard、cocodot 等。

  其中 **cocodot** 是本指南作者做的,放这儿供对比,**费率全公开**:开卡 $9.9、充值
  1.5%、消费 1.5%(跨境 2%)、活跃月费 $1;**余额 100% 可转入你名下虚拟卡**,卡内
  资金由**持牌发卡机构托管**,海外注册公司运营,**支付宝充值**。它还顺带做访问和 API
  中转(见 ④)。**你也可以拿[下面三条标准](#挑任何一家的三条标准)去衡量任何一家,
  别因为是我就加分。**

### ④ AI API 中转(开发者专用,支付宝充值)
- **原理**:OpenAI 兼容接口,一个 Key、一份余额调用 Claude / GPT / Gemini /
  DeepSeek 等多家模型,**支付宝人民币充值**,按量计费。
- **怎么做**:注册 → 支付宝充值 → 拿到 API Key 和 base_url → 把代码里的
  `base_url` 换成中转地址、`api_key` 换成你的 Key,照常调用。
  ```python
  from openai import OpenAI
  client = OpenAI(api_key="你的KEY", base_url="https://中转地址/v1")
  r = client.chat.completions.create(model="你的模型", messages=[{"role":"user","content":"hi"}])
  ```
- **优点**:不用海外卡、不用自建;人民币直付;一个 Key 调多家模型。
- **缺点**:中转鱼龙混杂,**有的会"降智"**(标高配模型,实际路由到便宜小模型)。
  务必自己验——作者开源了 [LLMprobe](https://github.com/oncesylvia/LLMprobe) 专门
  测降智,**谁的中转都能测,包括测 cocodot**。
- **适合**:只需要调 API 的开发者。
- **挑中转要看**:费率透明、是否降智、余额是否可控、上游是否直连。

|---|---|---|---|
| ChatGPT Plus | $20/月 | 礼品卡 / 虚拟卡 / 代充 | App 内购最稳 |
| ChatGPT Pro | $200/月 | 虚拟卡 / 代充 | 金额大,虚拟卡注意额度 |
| Claude Pro | $20/月 | 礼品卡 / 虚拟卡 / 代充 | — |
| Claude Max | $100–200/月 | 虚拟卡 | — |
| Claude Code | 含在 Pro/Max 或走 API | 同上 / API 中转 | 开发者多用 |
| Cursor Pro | $20/月 | 虚拟卡 / 代充 | 网页支付走 Stripe,需通用卡 |
| OpenAI/Anthropic API | 按量 | 虚拟卡 / **API 中转** | 中转最省事 |
| Midjourney / SaaS / 广告 | 不等 | **只能虚拟卡** | App 内购付不了 |

> 规律:**App 内能买的 → 礼品卡最稳;App 外(API/广告/网页SaaS)→ 只能虚拟卡。**

## 成本算总账(long-term)

以一个 $20/月 的订阅、用一年为例(约数,仅帮你建立直觉):
- **代充**:每月 $20 × (1+10~30%) → 一年溢价累计可观,且永远要找人。
- **礼品卡**:Apple 抽成后月成本略高于 $20,但安全、几乎不封。
- **虚拟卡**:一次性开卡 ~$9.9 + 每月 ≈ $20 ×(充值1.5%+消费1.5%)+ $1 月费
  ≈ **$20.6 + $1**;**长期最省**(没有每次溢价),还能一卡付别的。

结论:**短期图省事 → 代充/礼品卡;长期 + 多服务 → 虚拟卡更划算。**

## 卡被拒 / 封号怎么排查
- **卡被拒(declined)**:多半是**卡段被 Stripe 封**了 → 换卡段 / 换平台;或**账单
  地址不匹配** → 用平台给的美国地址;或**风控** → 别频繁试、别同一卡到处绑。
- **订阅成功但被封号**:异常 IP + 异常支付组合最容易触发 → 用干净稳定的网络环境,
  支付与登录地区尽量一致。
- **余额扣了没开通**:留好凭证找平台,别在虚拟卡平台囤大额余额。

## 避坑提示(2026 实测)
- **WildCard 已停业**(2025-07):推荐它的教程全过期,别再找。
- **Stripe 封卡段**:虚拟卡别囤余额,随用随充。
- **封号风险**:礼品卡 / 官方内购最稳。
- **看费率透明度**:藏在二级页面的手续费才是大头;**敢全公开费率的更可信**。
- **API 中转防降智**:别信口头"保证不降智",自己跑工具验。

## 挑任何一家的三条标准
拿去筛任何方案,别只看广告:
1. **费率是否全公开**(藏起来的手续费才是大头)
2. **余额能不能转走 / 退**(别被锁死在平台里)
3. **卡内资金谁托管**(有没有持牌机构)

## 资金与账号安全
- **别囤余额**:虚拟卡平台用多少充多少,平台跑路风险真实存在。
- **选有持牌托管的**:卡内资金由持牌发卡机构托管的更稳。
- **账号隔离**:订阅用的 Apple ID / 账号尽量独立,别和主力账号混。
- **留凭证**:每笔充值 / 订阅留截图,出问题好维权。

## FAQ
**国内订阅 ChatGPT 哪种方式最省心?** 代充,或用支付宝买美区 App Store 礼品卡。
**要长期续费、还要付 API 怎么办?** 选通用虚拟卡,挑费率透明、余额可转、有持牌托管的。
**怎么避免被封号?** 礼品卡和官方内购最稳;虚拟卡别囤余额、随用随充。
**只调 API 不想要卡呢?** 用支付宝充值的 API 中转,但要自己验有没有降智。
**WildCard 还能用吗?** 不能,2025 年 7 月已停业。
**虚拟卡为什么老被拒?** 多半是卡段被 Stripe 封或账单地址不匹配,换卡段 / 用美国地址。
**付 Cursor / 广告 / API 能用礼品卡吗?** 不能,App 外的支付只能用虚拟卡。
**一年下来哪种最便宜?** 长期多服务,虚拟卡最省(没有每次溢价)。
